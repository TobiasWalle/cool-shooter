# Cool Shooter

A simple couch/network Shooter with simple graphics, simple weapons and awesome sounds. Mainly a sandbox to try out unity and networking.

## Features

### V1

* 1st Person Shooter
* Instagib
* Networking
* Jumping
* Fast Pace
* Respawn
* Simple Level

### V2

* Different Weapons
* Sp(l)itScreen
* PowerUps
* Ammunition
* Head-Shot

### V3

* Grappling Hooks

### V4

* Stupid Bots
* Wall Jumping
* Super Mario Kills
